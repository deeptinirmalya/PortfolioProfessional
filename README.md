A sleek, modern, and highly interactive professional portfolio website built with a focus on clean design and smooth user experience. This portfolio showcases projects, skills, and the professional journey of an aspiring AI/ML Engineer.

## ✨ Features

- **Premium Modern Design**: A minimal and professional aesthetic using **Inter** and **Playfair Display** typography.
- **Dynamic Animations**: Powered by **GSAP (GreenSock)** for smooth entrance reveals and scroll-triggered transitions.
- **Responsive Layout**: Fully optimized for all devices, from high-resolution desktops to mobile screens.
- **Project Showcase**: A detailed project section with alternating layouts, showcasing machine learning and backend work.
- **Interactive Journey**: A visual timeline tracing the educational and professional path.
- **Functional Contact Form**: Integrated with **Formspree** for direct email delivery without a backend.
- **Dynamic Navigation**: Auto-highlighting navigation links based on scroll position and a "Back to Top" functionality.

---

## 🛠️ Built With

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Animation**: [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform)
- **Plugin**: GSAP ScrollTrigger
- **Forms**: [Formspree](https://formspree.io/)
- **Fonts**: Google Fonts (Inter & Playfair Display)

## 🔧 Installation & Usage

1. **Clone the repository**:
   ```bash
   git clone https://github.com/deeptinirmalya/PortfolioProfessional.git
   ```

2. **Open the project**:
   Simply open `index.html` in any modern web browser.

3. **Customization**:
   - Update your info in `index.html`.
   - Update your contact form ID in `index.html` (look for the `action` attribute in the `<form>` tag).
   - Modify styles in `style.css` and animations in `main.js`.
